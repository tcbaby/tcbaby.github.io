Git 是一个开源的分布式版本控制系统，用于敏捷高效地处理任何或小或大的项目。

git工作区域

  • 工作区(Working Directory): 添加、编辑、修改文件等操作
  • 暂存区： 暂存修改的文件，最后统一提交到git仓库
  • git仓库(Git Repository)：最终确认的文件保存到仓库，成为一个新的版本
