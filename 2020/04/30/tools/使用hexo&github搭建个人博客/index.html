搭建个人博客

本地搭建blog

  1. 在本地初始化博客项目

  2. 设置博客标题和博主：打开配置文件_config.yml ，找到下面配置

  3. 使用hexo server命令，启动博客项目。访问http://localhost:4000/即可看到我们的博客。

GitHub Page

  1. 在GitHub上创建一个用户名.github.io的仓库

  2. 在_config.yml文件中指定仓库地址

  3. 生成静态文件并推送到GitHub仓库

  4. 访问https://用户名.github.io即可。
